# 📧 Smart Email Summarizer

An AI-powered web application that summarizes long emails using the DistilBART Transformer model.

## ✨ Features

- 📝 AI Email Summarization using DistilBART
- 📂 Upload TXT or PDF emails
- 🏷️ Keyword Extraction
- 😊 Sentiment Analysis
- 👤 Named Entity Recognition
- 📊 Email Statistics
- 📥 Download Summary

## 🚀 Live Demo

[Live App](https://your-app-url.streamlit.app)

## 🛠️ Technology Stack

- **Frontend**: Streamlit
- **Model**: DistilBART (Hugging Face Transformers)
- **NLP**: spaCy
- **Sentiment Analysis**: DistilBERT
- **Language**: Python 3.10+
